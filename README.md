# Algoritmos e Estruturas de Dados I

Repositório destinado ao armazenamento de exercícios, trabalhos, implementações e estudos desenvolvidos durante a disciplina de Algoritmos e Estruturas de Dados I.

A disciplina tem como foco o desenvolvimento de técnicas fundamentais de programação, análise de algoritmos e implementação de estruturas de dados clássicas, utilizando diferentes abordagens para resolução de problemas computacionais.

## Conteúdos Abordados

- Recursão
- Busca Linear e Busca Binária
- Análise de Complexidade
- Algoritmos de Ordenação
  - Bubble Sort
  - Selection Sort
  - Insertion Sort
- Backtracking e Enumeração
- Tipos Abstratos de Dados (TADs)
- Listas Ligadas
  - Simples
  - Duplamente Ligadas
  - Circulares
- Pilhas e Filas
- Matrizes e Listas Ortogonais
- Árvores Binárias
- Árvores Binárias de Busca
- Filas de Prioridade
- Heaps
- Técnicas Algorítmicas
  - Divisão e Conquista
  - Programação Dinâmica
  - Algoritmos Gulosos
  - Busca em Largura

## Objetivos da Disciplina

- Desenvolver raciocínio algorítmico
- Implementar e manipular estruturas de dados clássicas
- Compreender eficiência e complexidade de algoritmos
- Resolver problemas utilizando diferentes paradigmas algorítmicos
- Aprimorar habilidades de programação e modelagem computacional
